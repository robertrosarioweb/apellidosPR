## Overview

This project uses **vanilla JavaScript**, **modern ES modules**, and lightweight libraries for building interactive, data-driven web pages.
This is an interactive data-driven web page for analysing data of surname concentration in Puerto Rico.

### Key Points

- **JavaScript / DOM Manipulation:** The UI is built using plain JavaScript and direct DOM manipulation (`document.createElement`, `appendChild`, `style` properties).  
- **CSS Styling:** Styling is applied both via inline JS (`element.style`) and a `<style>` tag to customize page appearance.  
- **ES Modules:** Modules are imported using ES module syntax (`import * as aq from "npm:arquero"`) from npm or CDN, allowing modular, modern JavaScript development.  
- **Libraries Used:**
  - **[Arquero](https://uwdata.github.io/arquero/):** Data manipulation library for handling tabular data, similar to pandas in Python.  
  - **[Observable Inputs](https://cdn.jsdelivr.net/npm/@observablehq/inputs/):** Library for interactive UI components and controls, often used in data visualization notebooks.  

> **Note:** This project does **not** use a traditional front-end framework like React, Vue, or Angular; it is a lightweight, library-based interactive setup.

## How to use
- Download the folders
- Open terminal in file inded.md
- Type "npm run dev"
